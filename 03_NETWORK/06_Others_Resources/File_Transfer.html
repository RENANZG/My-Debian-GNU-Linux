File Transfer Tools Comparison

Tool Description Installation Copying Preserves Permissions Progress Display
RSYNC
Arch Wiki		 Sync files locally or remotely. 
$ sudo apt install rsync
 
$ rsync -ah --progress ~/source /dest
 Yes Yes
DD
 Copy and convert files. Included by default 
$ dd if=~/source of=/dest status=progress
 Yes Yes
PV
 Monitor data progress. 
$ sudo apt install pv
 
$ pv ~/source > /dest
 No Yes
CP
 Copy files and directories. Included by default 
$ cp -a ~/source /dest
 Yes (with -a) No
WGET
 Download files from web. 
$ sudo apt install wget
 
$ wget -O /dest http://example.com/source
 No Yes
CURL
 Transfer data with URLs. 
$ sudo apt install curl
 
$ curl -o /dest FILE://source
 No No
SCP
 Secure copy protocol. 
$ sudo apt install openssh-client
 
$ scp ~/source user@remote:/dest
 Yes No
RCLONE
 Manage cloud storage. 
$ sudo apt install rclone
 
$ rclone copy ~/source remote:/dest
 Yes Yes
BBCP
 High bandwidth network transfer. 

# Download and build
$ wget http://www.slac.stanford.edu/~abh/bbcp/bbcp.tgz
$ tar -xvzf bbcp.tgz
$ cd bbcp/src
$ make
$ sudo cp bin/bbcp /usr/local/bin/
                
$ bbcp ~/source user@remote:/dest
 No Yes
